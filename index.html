L'application qui va vous facilitez votre visite à Paris

MyBench est une application qui vous permet de trouver rapidement et facilement un banc proche de votre géolocalisation pour vous permettre de déguster votre déjeuner ou de prendre une pause dans la plus belle ville du monde !

Amazing Features

Trouvez votre petit coin de paradis !

MyBench vous permet de vous géolocaliser automatiquement et de vous suggérez tout les bancs proches de votre position sur une carte avec diverses informations tels que : la pollution,l'environnement aux alentours et le bruit.

Pour vous facilitez la vie

MyBench vous propose aussi une liste de bancs les plus proches pour vous facilitez votre recherche.
